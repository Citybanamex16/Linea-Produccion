# Linea-Produccion
Simular una fabrica donde productos pasen por etapas: ensamblado, verificado y empaquetado. Estos procesos serán ejecutados por maquinas. Cada maquina tiene su comportamiento y velocidad de operación. Podemos inducir errores y calcular la eficiencia de la linea. El usuario decidirá el tipo y cantidad de productos a crear. Pensado para poder ser ejecutado en Terminal.
